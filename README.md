# 🐞 GitHub Issue Tracker

✨ A simple and modern GitHub Issue Tracker web application built with React that allows users to search, view, and explore GitHub repositories and their issues in a clean and interactive interface.

This project demonstrates real-world API integration using the GitHub API and dynamic UI rendering.

---

## 🚀 Live Demo
🌐 https://khanm206.github.io/gitHub-issue-traker

---

## 🛠️ Tech Stack

🟨 JavaScript (ES6+)  
🎨 Tailwind CSS  

---

## ✨ Features

🔎 **GitHub Repository Search**  
- Search any public GitHub repository by name

🐞 **Issue Listing System**  
- View all issues of a selected repository

📄 **Issue Detail View**  
- Open individual issues with full details

🌐 **Real-time API Integration**  
- Fetches live data from GitHub REST API

🎯 **Smooth UI Experience**  
- Clean navigation and responsive layout

⚡ **Fast & Lightweight**  
- Optimized API calls using Axios
